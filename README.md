# Typst Resume

## Install Typst

Follow the installation instructions at https://github.com/typst/typst#installation

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/fshrr/typst-resume.git
   cd typst-resume
   ```

2. Add your resume `.typ` file in the `resumes/` folder (use `ML.typ` as an example)

3. Make the generate script executable:
   ```bash
   chmod +x generate.sh
   ```

4. Run the executable with your name:
   ```bash
   ./generate.sh "John Doe"
   ```
